# 🩵 Blue Clínica — Landing Page

> **Desenvolvimento com ciência. Cuidado que acolhe.**

Landing page institucional da **Blue Clínica**, clínica multidisciplinar especializada em neurodesenvolvimento infantil, com atendimento baseado em evidências para crianças, adolescentes e famílias no Maranhão.

---

## 🌐 Acesso

- Site oficial: [clinicablue.com.br](https://clinicablue.com.br/)
- Instagram: [@clinicablue.itz](https://instagram.com/clinicablue.itz)

---

## 🏥 Sobre a Clínica

Fundada em **06/10/2020**, a Blue Clínica oferece atendimento multidisciplinar baseado em evidências para promover desenvolvimento, autonomia e qualidade de vida.

**Unidades no Maranhão (MA):**
- 📍 Imperatriz
- 📍 São Luís
- 📍 Açailândia
- 📍 Balsas

## 🗂️ Estrutura do Projeto

```
Blue Clínica landing page/
├── index.html          # Página principal (HTML + CSS + JS inline)
├── ds-base.js          # Design system base
├── image-slot.js       # Componente de imagens
├── assets/             # Imagens, fontes e recursos estáticos
├── uploads/            # Arquivos de mídia enviados
├── screenshots/        # Capturas de tela do projeto
├── scraps/             # Arquivos auxiliares / rascunhos
└── _ds/                # Design system (tokens e estilos base)
```

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura da página |
| **CSS3** | Estilização com variáveis de design (tokens), gradientes e animações |
| **JavaScript** | Interatividade e animações |
| **GSAP + ScrollTrigger** | Animações avançadas com scroll |
| **Lenis** | Scroll suave |
| **Schema.org (JSON-LD)** | SEO semântico para negócio médico (`MedicalBusiness`) |
| **Google Fonts** | Tipografia (Plus Jakarta Sans + Inter) |

---

## ✨ Características Técnicas

- Design 100% responsivo (mobile-first)
- CSS com variáveis de design tokens (paleta azul + tipografia)
- Efeitos de glassmorphism e backdrop-filter
- Animações com keyframes (pulse, marquee)
- SEO otimizado com meta tags Open Graph e Schema.org
- Sem dependência de frameworks CSS externos

## 📁 Como rodar localmente

Como é um projeto HTML puro, basta abrir o arquivo no navegador:

```bash
# Clone o repositório
git clone https://github.com/Thiagocastro1997/BLue_site.git

# Entre na pasta
cd BLue_site

# Abra no navegador (Windows)
start index.html

# Ou use o Live Server do VS Code para melhor experiência
```
